# Architecture Decision Records

This directory contains the architectural and engineering decisions made in this project, documented using a lightweight version of the [Nygard ADR format](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).

Each ADR captures the context that existed at the time of the decision, the decision itself, and its consequences — including trade-offs. The goal is not to justify every choice but to make the reasoning visible, so future contributors (or interviewers) can understand why the project is shaped the way it is.
